# Unfold

Modern TypeScript + Vite project with React, TailwindCSS, and ESLint.

## Quick Start

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```
